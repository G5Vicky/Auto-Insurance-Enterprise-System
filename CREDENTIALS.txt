QUICK START (No MySQL needed — runs on H2 in-memory DB):
  1. cd autoinsurance
  2. mvn spring-boot:run    (or)    ./mvnw spring-boot:run
  3. Open http://localhost:8080
  4. Login with any credentials above

To use MySQL instead:
  - Edit application.properties → spring.profiles.active=mysql
  - Create database: CREATE DATABASE insurancedb;
  - Set credentials in application-mysql.properties
